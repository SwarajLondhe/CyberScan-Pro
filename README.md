# CyberScan Pro 🛡️
A professional Vulnerability Scanner mini-project built with Python (Flask) and Web Technologies.

## 🚀 Features
- **Port Scanning:** Checks for open ports (FTP, SSH, HTTP, HTTPS).
- **Vulnerability Logic:** Identifies insecure protocols and suggests fixes.
- **Risk Assessment:** Categorizes targets as LOW, MEDIUM, or HIGH risk.
- **Hacker UI:** A modern, terminal-style dark theme interface.

## 🛠️ Tech Stack
- **Backend:** Python 3, Flask, Socket Programming
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
